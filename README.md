## 📖 THE CABIN BRIEFING (GAME OVERVIEW)

A catastrophic merge conflict has fragmented the cloud databases! **GitRPG** is a text-based rogue-like RPG engine built directly into your GitHub environment using **100% Vanilla Java (`java.util.*`)**. 

You do **not** run this game on your laptop. You play this game by piloting **GitHub Actions workflows** via dropdown macro panels! Battle automated glitches, execute tactical code interventions, and check the runner matrix streams for your real-time ASCII combat evaluation score.

## 🎮 HOW TO PLAY (INSERT COIN & START)

### STEP 1: HIJACK THE SOURCE FILE (FORK & CLONE)
To open up your own dedicated local arcade machine, click the **FORK** button at the top-right of this repository, then clone your copy down to your machine:
```bash
git clone https://github.com
```

### STEP 2: ACTIVATE ACCELERATOR NODE (GITHUB ACTIONS)
1. Navigate to the **Actions** tab on your forked GitHub repository page.
2. Select **⚔️ Play GitRPG Terminal Dungeon** from the left-hand menu.
3. Click the **Run workflow** dropdown configuration console on the right side.

Fill out your target character variables before launching into the runtime warp:
